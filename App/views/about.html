How we've got the data?

  1. Crawler

    2. We have created a crawler that sends requests to the API Riot Games with all the MatchId provided of EUWest server, of which 19,576 have been possible to obtain items for patch 5.11 and 19,821 for 5.14.

    Crawler in github

  2. Formatter

    3. This Crawler-Formatter (created by us) reads and decodes the results to extract all the data we want to use a json file format.

    Files:

    1. Json 5.11
    2. Json 5.14

  3. APP

    4. The application was created using technologies like Angularjs and Bootstrap. This application has 3 main sections:

    1. AP Items: This section compares the items of hability power(AP) in different versions of the game with powerful filters.
    2. Champions: An impressive section that can compare from a single view, the main impact of the different versions of the game champions.
    3. Champions detaills: Specializes in analyzing in great detail the changes that have been the champions and their respective objects from one version to another version
